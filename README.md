# Donauinsel Pontoon Comfort Dataset

This repository contains an updated POI and polygon dataset about pontoons on Donauinsel in Vienna, Austria.

The dataset adds information about shade availability, lawn/mowed status, mapped chill areas, field photos, and distances to sanitary and safety facilities.

## Data

- `data/pontoons.geojson`
- `data/chill_areas.geojson`
- `data/facilities.geojson`

## Use Cases

- Find shaded pontoons during hot days.
- Find pontoons with mowed lawn and nearby sanitary facilities.
- Compare recreational comfort and safety of different pontoon areas.
