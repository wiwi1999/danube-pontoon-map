# Donauinsel Pontoon Comfort Dataset

This repository contains an updated and extended dataset about pontoons on the Donauinsel in Vienna, Austria.

The dataset is based on the open government dataset "Donauinsel Freizeiteinrichtungen Standorte Wien" and was enriched with field observations about shade availability, lawn/mowed status, mapped chill areas, field photos, and distances to sanitary and safety facilities.

## Data

## Main Added Attributes

- shade availability
- lawn/mowed status
- chill-area polygons
- chill-area size
- distance to sanitary facilities
- distance to safety facilities
- field photos

## Use Cases

- Find shaded pontoons during hot days.
- Find pontoons with mowed lawn and nearby sanitary facilities.
- Compare recreational comfort of different pontoon areas.
